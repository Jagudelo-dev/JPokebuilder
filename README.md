# Pokémon Team Builder (5ª Gen)

Herramienta web para construir equipos Pokémon competitivos con soporte 
de asistente para crear equipos mas facil mente y movimientos hasta la 9ª.

## 🚀 Demo en vivo

[Pruébalo aquí](https://tuusuario.github.io/jpokebuilder)

## ✨ Funcionalidades

- Equipo de 6 Pokémon completamente personalizable.
- Selección de movimientos por nivel, huevo y MT/MO.
- Ajuste de IVs, EVs, naturaleza, nivel, habilidad, objeto y apodo.
- Cálculo automático de estadísticas totales.
- Gráfico radar de stats base con promedio del equipo.
- Tabla de resistencias, debilidades, inmunidades y neutralidades.
- Calculador de coberturas ofensivas por tipos.
- Asistente táctico con consejos de sinergias y debilidades.
- Exportación/importación en formato Showdown.
- Persistencia automática en el navegador.
- 151 Pokémon de Kanto (ampliable mediante JSON).

## 🛠️ Tecnologías

HTML5, CSS3, JavaScript vanilla (sin frameworks), datos desde PokeAPI.

## 📦 Estructura del proyecto

- `js/core/` – Lógica de combate, tipos, cobertura.
- `js/data/` – Datos de Pokémon, movimientos, objetos.
- `js/io/` – Persistencia y exportación/importación.
- `js/ui/` – Interfaz de usuario.
- `tools/` – Scripts para generar datos automáticamente.
